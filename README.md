# Ambiance Café Website
Website for Ambiance Café, where good coffee meets good vibes.

## Overview:

### Home Page:
- Hero carousel with 3 rotating slides
- Best sellers showcase (3 featured items)
- Three pillars section (Ambiance, Coffee, Vibe)
- Café introduction
### Menu Tab: 
- Searchable menu
- Sidebar navigation with categories and subcategories
- Categories:
  - **Coffee**: Espresso-Based, Brewed, Cold Brew
  - **Non-Coffee**: Tea & Fruit Teas, Milk-Based, Refreshers
  - **Pastries**: Cakes & Slices, Baked Goods
  - **Others**: Merchandise
### About Us:
- Our Story section with image
- Core Values (Quality & Craftsmanship, Ambiance & Experience, Community & Connection)
- Mission & Vision statements
### Contact us:
- Social media links (Facebook, Instagram, Twitter)
- Email address
- Phone numbers
### Join our Crew:
- Full Name
- Email Address
- Mobile Number
- Position Applied
- Message
- Attach File (CV)

## Features:

**Hero Carousel**: Homepage slider with multiple images and animation.
**Interactive Menu**: Searchable menu with category filtering and organized sections
**Sticky Navigation**: Always-accessible navigation bar
**Transition and Hover Effects**: Enhanced user experience.
**Social Media**: Direct links to Facebook. Instagram, Twitter by clicking on their icons.
**Contact Information**: Email and Phone numbers are easy to read.

## Technologies: 

- **HTML5**
- **CSS3**
- **JavaScript**
- **Bootstrap 5.3.0**: Carousel component and responsive utilities
- **Bootstrap Icons**: Icon library for UI elements
- **Animate.css**: CSS animation library
- **Google Fonts**: Montserrat font family
